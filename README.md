# Personal Portfolio Website

This project is a personal portfolio website developed using **Next.js** and styled with **TailwindCSS**, following the ByteGrad tutorial. The site incorporates smooth animations and interactions powered by **Framer Motion** to create a visually engaging and interactive experience.

![Portfolio Screenshot](https://github.com/user-attachments/assets/37efd7e0-c79b-41a0-84ed-b4ae2e0a8caa)

## Features

- **Responsive Design**: Fully responsive and optimized for all screen sizes and devices.
- **Smooth Animations**: Uses `Framer Motion` for sleek, fluid transitions and interactive elements.
- **Modern UI/UX**: Clean, minimal, and modern user interface built with TailwindCSS.
- **Next.js Optimized**: Leverages the power of Next.js for server-side rendering and SEO-friendly pages.

## Built With

- **Next.js**: A React framework that enables server-side rendering and static site generation.
- **TailwindCSS**: A utility-first CSS framework for fast and responsive UI development.
- **Framer Motion**: A powerful animation library for React to bring the site to life with animations.
